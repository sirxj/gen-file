## gen-file-cli
  quickly create template files

## Install
```bash
  npm install gen-file-cli -g
```

## Usage
```shell
  gen-file salesMange salesOrder '销售订单'
```

## Samlpe `genfile.js`
```js
  module.exports = {
    outPath: '', // default src/view
    templatePath: '' // default template
  }
```
